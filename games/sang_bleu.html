« Sang » bleu

Afficher/masquer les instructions

Principe d'une fiction interactive :
Cet écrit est une fiction interactive : Une description présente un lieu, les objets et personnages qui s’y trouvent et avec lesquels il est possible d’interagir. En dessous de la zone de description, se trouve un prompt (« > »). En face du prompt, vous allez pouvoir commander des actions et des interactions avec les différents objets et personnages évoqués dans cette fiction. L’interpréteur peut comprendre bon nombre d’actions décrites par un verbe et, souvent, impliquant un objet : Par exemple, vous pouvez écrire : « prendre la pelle », « creuser ceci », « regarder au ciel » ou « parler à untel ». Lorsque vous validerez l’action en appuyant sur la touche « entrée », le programme retournera une réponse contextuelle.

Déplacement :
Typiquement, dans une fiction interactive comme dans celle ci-dessous, on utilisera les points cardinaux pour se déplacer. On peut alors écrire dans la barre de commandes « aller » suivit d’une direction valide : « nord », « sud », « est » ou « ouest », avant d’appuyer sur la touche « entrée » pour valider. Cela donnera, par exemple : « aller au nord ».

Inventaire :
Consultez l’inventaire en entrant : « INVENTAIRE » dans la barre de commande.

Liste non exhaustive des actions possibles :
regarder... aller... parler à... prendre... ouvrir... fermer... lire... utiliser... pousser... lécher... équiper... toucher... sentir... tourner... dormir... insérer *objet 1* dans *objet*... creuser... actionner... manger... entrer dans... sortir de... embrasser... écouter... sentir...

Conseil :
Les verbes impliquant les sens permettent d’obtenir des indices importants pour comprendre le monde décrit dans cette fiction : Regardez, touchez, sentez, léchez... Tous les objets cités sont sujets à examen par les sens. Interagissez de manières variées avec les décors pour développer la narration.
